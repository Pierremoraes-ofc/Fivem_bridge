## 🔗 FiveM Bridge System

### 🇺🇸 English

A powerful and flexible **Bridge System for FiveM** that allows seamless integration between multiple frameworks such as **QBCore**, **ESX**, and standalone resources.

It abstracts framework-specific functions into a unified interface, making your scripts cleaner, scalable, and framework-independent.

🚀 Perfect for developers who want to build reusable and professional systems.
